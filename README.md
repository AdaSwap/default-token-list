# AdaSwap default token list

Forked from the [Uniswap default-token-list](https://github.com/Uniswap/default-token-list).


### Before build

* Run `yarn install`

### Build

* Run `yarn build`

### Publish to npm 
 * Change **name** and **version** in package.json file
 * Run `npm publish --access public`
